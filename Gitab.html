JavaScript 数组操作

语言

JavaScript

标签

描述

一些常用的JavaScript数组操作函数，包括去重、扁平化和求和。

代码内容

// 数组去重
function uniqueArray(arr) {
    return [...new Set(arr)];
}

// 数组扁平化
function flattenArray(arr) {
    return arr.flat(Infinity);
}

// 数组求和
function sumArray(arr) {
    return arr.reduce((acc, val) => acc + val, 0);
}
